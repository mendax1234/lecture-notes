```java
class Solution {
    public boolean judgeCircle(String moves) {
        int x = 0, y = 0;
        for (char m : moves.toCharArray()) {
            if (m == 'R') {
                ++x;
            } else if (m == 'L') {
                --x;
            } else if (m == 'U') {
                ++y;
            } else {
                --y;
            }
        }
        return (x == 0 && y == 0);
    }
}
```

The learning point for this problem is how to do the String Iteration using `.toCharArray()`.
