# Delay Calculation

Synopsys supports several delay models. These include

1. the CMOS generic delay model,
2. CMOS piecewise linear delay model and
3. the CMOS non-linear table lookup model.

The **non-linear** model is the most prevalent delay model used in the ASIC world.
