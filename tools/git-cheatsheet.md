# Git CheatSheet

## Useful Git Skills

| Skills       | Description                                                                                                                                                                    |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `git commit` | To enter a short detailed description about a certain commit. Use `git commit -m "Commit message \n \n Start description"\n`. (replace `\n` with <kbd>Enter</kbd> when typing) |
