# Introduction

{% hint style="warning" %}
This book is intended to serve as a **supplement** to a more theoretical textbook such as [Digital Integrated Circuits: A design Perspective](../textbook-1-dicadp/introduction/).
{% endhint %}

The design process for digital integrated circuits is extremely complex. This is why we need to use tools to help. Finding a combination of tools and a way of using those tools that works for a particular design is know as finding a "_tool path_" for that project. This book is an attempt to codify at least one working _tool path_ for a Cadence/Synopsys flow that students and researchers can use to design digital integrated circuits.

The tools and flows described here generally fall within the category of supporting ASIC design with **standard library cells**.

{% hint style="success" %}
In this book, I'll call the tools "CAD Tools" and the companies "EDA companies".
{% endhint %}
